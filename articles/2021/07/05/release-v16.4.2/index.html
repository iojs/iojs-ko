Node v16.4.2(현재 버전)

  • 작성자: Bethany Nicolle Griggs
  • 원문: Node v16.4.2 (Current)
  • 번역: outsideris

주요 변경사항

Node.js 16.4.1에서 발생한, 영어가 아닌 로케일에서 Windows 설치 프로그램에 생긴 회귀 문제를 이번 릴리스에서 수정했습니다.
Windows 설치 프로그램을 사용하지 않으면 이번 릴리스를 다운로드받을 필요가 없습니다.
