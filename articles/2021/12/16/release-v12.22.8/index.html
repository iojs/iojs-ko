Node v12.22.8(LTS)

  • 작성자: Richard Lau
  • 원문: Node v12.22.8 (LTS)
  • 번역: outsideris

주요 변경사항

이번 릴리스에는 회귀 오류를 수정하는 c-ares 업데이트가 포함되어 있습니다. 이 회귀 오류는 Node.js
12.22.5에서 언더스코어가 포함된 CNAME 레코드를 처리하다가 도입되었습니다.
#39780.

Mozilla’s Network Security Services 3.71의 인증서로 루트 인증서를 업데이트했습니다.
#40281.
