Node v14.15.3(LTS)

  • 작성자: Bethany Nicolle Griggs
  • 원문: Node v14.15.3 (LTS)
  • 번역: outsideris

주요 변경사항

Node.js v14.15.2에는 요청 객체를 클론할 때 보고된 손상을 일으킨 커밋이 포함되어 있었습니다. 이번 릴리스는 이 동작을 일으킨 커밋을 되돌립니다. 자세한 내용은 https://github.com/nodejs/node/issues/36550을 참고하세요.
