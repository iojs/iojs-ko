주요 변경사항

이번 릴리스는 보안 릴리스입니다.

아래 취약점을 수정했습니다.

  • CVE-2020-8201: CR을 하이픈으로 변환해서 발생하는 HTTP 요청 스머글링(Smuggling)(높음).
  • CVE-2020-8252: fs.realpath.native가 버퍼 오버플로우를 유발할 수 있습니다.(중간)
