Node v14.11.0(현재 버전)

  • 작성자: Richard Lau
  • 원문: Node v14.11.0 (Current)
  • 번역: outsideris

주요 변경사항

이번 릴리스는 보안 릴리스입니다.

아래 취약점을 수정했습니다.

  • CVE-2020-8251: 끝나지 않은 HTTP/1.1 요청때문에 리소스 소진 CWE-400에 의한 서비스 거부(위험).
  • CVE-2020-8201: CR을 하이픈으로 변환해서 발생하는 HTTP 요청 스머글링(Smuggling)(높음).
