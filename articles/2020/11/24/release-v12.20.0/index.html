      https://github.com/nodejs/node/pull/35435

  • http:
    • (SEMVER-MINOR) http 에이전트에 스케줄링 옵션을 추가했습니다. (delvedor)

    • (SEMVER-MINOR) exports 패턴을 지원합니다. (Guy Bedford)
      https://github.com/nodejs/node/pull/34718

      https://github.com/nodejs/node/pull/35249
  • n-api:
    • (SEMVER-MINOR) 더 많은 기본 프로퍼티를 추가했습니다. (Gerhard Stoebich)
      https://github.com/nodejs/node/pull/35214
  • src:
    • (SEMVER-MINOR) node_contextify를 현대적인 THROW_ERR_*로 바꾸었습니다. (James M Snell)
      https://github.com/nodejs/node/pull/35470
    • (SEMVER-MINOR) node_process를 현대적인 THROW_ERR*로 바꾸었습니다. (James M Snell)
      https://github.com/nodejs/node/pull/35472
    • (SEMVER-MINOR) v8::Isolate 설정 콜백을 노출했습니다. (Shelley Vohr)
      https://github.com/nodejs/node/pull/35512
